# Suggested Commands

- Prefix shell commands with `rtk` in this environment.
- `npm run dev`: start Vite UI and CLI server against current repo.
- `npm run build`: build UI, server, and CLI outputs.
- `npm run lint`: run ESLint.
- `npm run test`: run Vitest once.
- `npm run release`: build and create npm package tarball.
- Runtime examples: `npx diffvision`, `diffvision --cwd /path/to/repo`, `diffvision main`, `diffvision --new-in feature --relative-to develop`, `diffvision-mcp --cwd /path/to/repo`.
